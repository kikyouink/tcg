# tcg-阵面对决

# 开始
```javascript
yarn
yarn serve
```

# 目前实现的功能
- 用户信息设置
- 全局插件，例如alert,strorage
- 在线匹配
- 卡牌构筑
- nodejs爬虫，自动爬取卡牌图片与卡牌信息
